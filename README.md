# Meet App

## Project Overview
This project aims to build a serverless, progressive web application (PWA) using React and a test-driven development (TDD) technique.
The app leverages the Google Calendar API to fetch upcoming events, providing users with a seamless experience to explore and visualize events.

## Key Features
1. **Filter Events by City**
2. **Show/Hide Event Details**
   - As a user, I should be able to click on an event so that I can view or hide event details.
3. **Specify Number of Events**
   - As a user, I should be able to specify the number of events to show so that I can control the amount of information displayed.
4. **Use the App When Offline**
   - As a user, I should be able to use the app when offline so that I can access information even without an internet connection.
5. **Add an App Shortcut to the Home Screen**
   - As a mobile user, I should be able to add the app to the home screen so that I can quickly access it like a native app.
6. **Display Charts Visualizing Event Details**
   - As a user, I should be able to view charts visualizing event details so that I can gain insights from the data in a graphical format.

## Technical Requirements
- The app must be a React application.
- The app must use the Google Calendar API and OAuth2 authentication flow.
